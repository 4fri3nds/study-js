1. Передать функцию как параметр

Функция, которая принимает параметром себя же

        

          let quoterDegry = function(num, func) { 
            return func(num); 
          }; 
          let quoter = function(x) { 
            return Math.pow(x, 2); 
          }; 
          
          // quoterDegry(2, quoter) - вторым параметром идет функция - без func()
          
          div( quoterDegry(quoterDegry(2, quoter), quoter) );

В качестве аргументов служат две функции

        
          function a() { return 3 }; 
          function b() { return 4 }; 
          function abc(a, b) { return a() + b() };

OR
