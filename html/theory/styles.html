     Свойства PRE
    
                  white-space: normal; // текст ведёт себя как обычно
                  white-space: nowrap; // Текст идёт одной строкой, переносов не происходит, за исключением вручную расставленных, например,тегом br.
                  white-space: pre; // Выводит текст с пробелами и без переносами
                  white-space: pre-wrap; // Выводит текст с пробелами и с переносами
                  white-space: pre-line; // Выводит текст с переносами, как было в коде html, но дополнительные пробелы игнорируются
