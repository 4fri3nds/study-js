    5. Continue
    Директива continue прекращает выполнение текущей итерации цикла. -- компилятор не идет дальше по условиям кода в текущей итерации. Метки для break/continue Бывает нужно выйти одновременно из нескольких уровней цикла. Например, внутри цикла по i находится цикл по j, и при выполнении некоторого условия мы бы хотели выйти из обоих циклов сразу: 
                            
                  for( key in item) { ... }

                  Пример
                  let str = 'hello world'; 

                  for (i in str) { 
                    if (str[i] == 'o') { 
                      continue 
                    } else { 
                      console.log(str[i]); //'hell wrld'
                    } 
                  }

    6. Switch-case
    Конструкция switch заменяет собой сразу несколько if. Она представляет собой более наглядный способ сравнить выражение сразу с несколькими вариантами. 
                    
                  Позволяет работать с переменными
                  var a = 1; 
                  var b = 0; 
                  switch (a) { 
                    case b + 1: 
                      alert( 1 ); 
                      break; 
                    default: 
                      alert('нет-нет, выполнится вариант выше') 
                    }

                  Можно положить несколько значений в один case
                    var a = 2+2; 
                    switch(a) { 
                      case 4: alert('Верно!'); break; 
                      case 3: // (*) 
                      case 5: // (**) 
                        alert('Неверно!'); 
                        alert('Немного ошиблись, бывает.'); 
                        break; 
                      default: 
                        alert('Странный результат, очень странный'); 
                      }
                      
                  Если в конце не ставить break то будут выполнятся все условия после правильного
                  switch (foo) { 
                    case 3: console.log(1) break 
                    case 4: console.log(2) break 
                    case 5: console.log(3) break 
                    default: console.log(4) break 
                  }

                  Нужен правильный тип данных. switch предполагает строгое равенство === 
                  var arg = prompt("Введите arg?")  //передается строка
                  switch (arg) {
                    case "2": 
                      alert( 'Выполнится 2' );
                    case 3: 
                      alert( '3 Никогда не выполнится' );
                  }
