Функции

  1. Функции в Javascript'е — немного странные штуки. Функции в Javascript'е — на самом деле объекты. А именно, объекты класса Function, создаваемые конструктором Function. В объекте Function содержится строка с JS-кодом данной функции.
  2. Первый пример вызовит функцию без клика 
              
            but.onclick = func(x, y); //вызов без клика
            but.onclick = func() { func(x, y) }; //вызов по клику

  3. обращаясь к методу arguments[i] мы можем передать любое кол-во параметров

              
            function display() {
              var z = 1;
              for(var i=0; i < arguments.length; i++) {
                z *= arguments[i];
                console.log(z);
              }
            };
            
            display(6); // 6
            display(6, 4) // 24
            display(6, 4, 5) // 120

  4. Вызов из анонимной замыкающей функции

              
            (function(){ 
              var title = "Hello World"; 
              function showTitle(){ 
                alert(title); 
              } 
              window.showSimpleTitle = showTitle; 
            })();
            
            showSimpleTitle();

  5. Вызов функций из обьекта

              
            var exampleObject = { 
              "title": "Example Title", 
              "showTitle": function(){ 
                alert(this.title); 
              } 
            };
            
            exampleObject.showTitle();

  6. Вызов функции в контексте обьекта

              
            function showTitle() { 
              alert(this.title); 
            } 
            
            var objectA = { 
              "title": "Title A", 
              "showTitle": showTitle 
            }; 
            
            var objectB = { 
              "title": "Title B", 
              "showTitle": showTitle 
            };
            
            objectA.showTitle(); 
            objectB.showTitle();
  7. Вывод первого задания Итого: если функция вызвана как свойство объекта, то this будет ссылаться на этот объект. Если вызывающего объекта нет, this будет ссылаться на глобальный объект window.
  8. Частая ошибка чтобы this в функции showTitle ссылалась на обьект exampleObject - нужно эту функцию запускать в контексте обьекта 
              
            exampleObject.showTitle - плохо
            exampleObject.showTitle() - хорошо
1. 1

This внутри функции и обьекта exampleObject будет указывать не на этот обьект а на глоб обьект window

из-за того, что в строке var showTitle = exampleObject.showTitle мы получаем ссылку только функцию, и при вызове showTitle() нет ссылки на исходный объект exampleObject, отчего this начинает ссылаться на объект window. 

        
          var title = "Global Title"; 
          var exampleObject = { 
            "title": "Example Title", 
            "showTitle": function(){ 
              alert(this.title);
            } 
          }; 
          
          var showTitle = exampleObject.showTitle; // вначале забираем ссылку на функцию
          showTitle(); // а тут вызываем функцию без ссылки на объект
